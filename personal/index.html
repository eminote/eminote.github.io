EMINOTE (source code hosted here on GitHub) is a project to build a secure, powerful, easy-to-use, cross-browser cryptography for large text in Javascript. Why Javascript? The ideea is not to have sensitive information available in some form on a server.

EMINOTE is secure. It uses the industry-standard AES algorithm at 128, 192 or 256 bits; the SHA256 hash function; the HMAC authentication code; the PBKDF2 password strengthener; and the CCM, OCB2 or GCM authenticated-encryption modes. Just as importantly, the default parameters are sensible (you can change any of this in settings, by the way): EMINOTE strengthens your passwords by a default factor of 1000 and salts them to protect against rainbow tables, and it authenticates every message it sends to prevent it from being modified. EMINOTE provides the best security which is practically available in Javascript.

EMINOTE is cross-browser. It was tested on all modern browsers (including various versions of Internet Explorer, Chrome, Firefox, Safari and Opera on Mac, Linux and Windows). If you find any failures, please report an issue on the below contact form.

Because the site is static (it saves nothing on the server) and a normal form would need some kind of database. For support Eminote uses Zoho as an external portal. You add an issue or question here.
